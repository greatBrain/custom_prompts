Contexto: Un sitema web LLM para la validación de documentos legales y contables usando IA/LLMs que detecte errores ortograficos, de redaccion, errores o fallas legales, clausulas abusivas, cosasa importantes faltantes, etc. Todo en el contexto de las leyes de Republica Dominicana. Puedes echarle un ojo a lo ya existente y hacer que el nuestro se diferencie.
Publico objetivo:Contadores independientes, pequenas firmas legales/contables, abogados, poblacion general.
Tecnologias frontend: HTML5, Vue3, Tailwind CSS 4 (o una version lo bastante actualizada que funcione).
Tecnologias backend: FastAPI, LLM, LangChain, Postgress. 
Tu trabajo: genera el código completo SI Y SOLO SI SOLO para el frontend utilizando para un MVP funcional de este sistema.
Objetivo: validar el producto, su aceptacion y su usabilidad. Debe ser usable, moderno, profesional, limpio e intuitivo, de diseño moderno y una paleta de colores suaves y agradables.
Debe reflejar confiabilidad y simplicidad.
Debes seguir el siguiente flujo logico para generar este proyecto:
1-Dame la instruccion o comandos, de manera secuencial y logica para crear el proyecto con npm y vue.
2-Define las dependencias a instalar.
3-Recomiendame en que archivo debo asignar cada pieza de codigo, cada modulo, donde va cada cosa.  
Debes entregar: 
1.Vista login/Registro minimalistas
2.Vista Principal (Dashboard): Mostrar un historial de análisis recientes con cards que representen cada documento analizado.
3.Vista Carga de Documentos:** Opción clara y prominente para subir documentos.

Web:

    * Paleta de colores moderna y armoniosa y algo llamativa.
